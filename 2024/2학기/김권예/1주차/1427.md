### reverse

- ##### .reverse(), reversed() 특징

```python
    # .reverse
    # 1.
    list_re = list.reverse()
    print(list_re) # None
    # 2.
    list.reverse()
    print(list) # 뒤집힌 리스트 반환
```

#### .reverse()

- #### 리스트에만 사용 가능
- #### 리스트를 뒤집는 것 뿐 리스트를 반환하지 않음

#### reversed()

- #### 리스트, 튜플, 스트링에 사용 가능
- #### 객체로 값을 반환
- #### ex. list(reversed(n)) / re_n = reversed(n)
