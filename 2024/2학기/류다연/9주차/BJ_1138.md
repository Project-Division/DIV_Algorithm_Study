# 백준 1138_한 줄로 서기

- ## 문제
    - ### [링크](https://www.acmicpc.net/problem/1138)

<br>

- ## 코드
    ```python
    N = int(input())
    li = list(map(int, input().split()))

    res = [0]*N

    for i in range(N):
        cnt = 0 # 빈 자리 개수

        for j in range(N):
            if cnt == li[i] and res[j] == 0: # 빈 자리 개수가 같고, 그 자리가 비어있으면
                res[j] = i+1
                break
            elif res[j] == 0: # 비어있는 자리면
                cnt += 1

    print(' '.join(map(str, res)))
    ```
